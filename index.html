CSS Selectors

Selector

Specificity

Description

Syntax
HTML CSS
!important Highest A tag that can be used in CSS to override all other selectors. Should be avoided if possible N/A eg. color: blue; !important
id High A tag used in <div> blocks used to give certain areas specific characteristics. <div id="input"> #input { }
class medium A tag used in elements used to give certain areas specific characteristics. class="input"; .input { }
element Low HTML element tags can be referenced directly in CSS to assign visual characteristics. e.g <h1> h1 { }
